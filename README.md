## What I Build

## Current Focus

I am building toward a personal agent stack where tools, memory, projects, and messaging can work together without turning into a black box. The goal is practical: useful agents, clear boundaries, recoverable actions, and workflows that can be inspected later.

## Operating Principles

- Keep tools local when local is enough.
- Prefer durable notes over repeated rediscovery.
- Make automation observable.
- Build interfaces that are usable before they are flashy.
- Treat private data and external actions carefully.
